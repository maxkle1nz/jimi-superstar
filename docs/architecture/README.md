# Architecture Overview

ANOMALY is organized around these major planes:

- Kernel Plane
- Truth Plane
- Capability Plane
- Compatibility Plane
- Execution Plane
- Memory Plane
- Surface Plane
- Closure Plane

Core runtime domains:

- Kernel Domain
- Truth Domain
- Capability Domain
- Execution Domain
- Memory Domain
- Interface Domain

The runtime is event-backed, policy-centered, and provider-neutral.

Additional constitutional centerlines:

- `RETROBUILDER` is the build doctrine
- `Mandala` is the canonical agent-state and capsule contract
- `FieldVault` is the canonical sealed artifact and shard substrate
